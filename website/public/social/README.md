Social media assets for Ben Yaparim Meta automation.

How to use:

1. Put Facebook and Instagram images in this folder.
2. Use lowercase ASCII file names with hyphens.
3. Prefer .jpg or .png files.

Suggested naming:

- benyaparim-2026-06-08-facebook-1.png
- benyaparim-2026-06-08-instagram-1.png
- benyaparim-2026-07-07-facebook-1.png
- benyaparim-2026-07-07-instagram-1.png

URL pattern after deployment:

- https://benyaparimci.com/social/benyaparim-2026-06-08-instagram-1.png
- https://benyaparimci.com/social/benyaparim-2026-07-07-instagram-1.png
